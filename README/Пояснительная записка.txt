Пояснительная записка

NewDesmos
Авторы: Мокрушин Роман, Фролов Иван, Бережной Иван

Идея:
Создать веб приложение способное рисовать графики по заданным функциям (как у desmos.com) и возможность делиться
проектами с этими графиками во встроенной социальной сети

Реализация:
Создание frontend и backend частей проекта а также создание REST-API

Технологии:
Numpy - библиотека для работы с многомерными массивами
MathPlotLib - библиотека для рисования координатной плоскости и графиков

Описание проекта:
Генерация картинок(и возможно анимаций) графиков, заданных математическими формулами.
На одном графике могут отображаться несколько формул, в формулах можно использовать отдельно определённые переменные.
Регистрация пользователей: В личном кабинете пользователь может сохранять свои графики.
Т.к. графики задаются исключительно текстом формул, то пользователю не обязательно регистрироваться,
чтобы поделиться графиком: достаточно сгенерировать ссылку, по которой будет генерироваться график.
